🧳 TL-Agent：你的智能旅行秘书
像一位贴心秘书一样，为你规划高效、便捷、省时省力的个性化行程。

TL-Agent 是一个基于大模型（LLM）的旅游规划 AI Agent，专注于智能路径规划与时空优化。它不仅能理解你的需求，还能主动计算步行距离、交通耗时、景点开放时间等现实约束，为你精准推荐地铁附近的酒店和顺路的优质餐馆，打造真正“走最少路、看最多景”的旅行体验。

✨ 核心功能
智能行程规划

根据目的地、偏好与当前时间，自动生成从出发到返程的完整日程。

真实时空感知

自动注入当前日期与时间，避免“早上9点吃晚饭”类逻辑错误，支持“现在出发”“今晚安排”等自然语言指令。

步行友好路线

基于地图 API 计算实际步行距离与耗时，优先选择相邻景点，减少无效奔波。

酒店 & 餐厅智能匹配

推荐靠近地铁站的酒店 + 行程沿线的高分餐厅，兼顾便利性与体验感。

持久化会话记忆

支持多轮对话上下文记忆，随时调整计划，像真人秘书一样持续协作。
